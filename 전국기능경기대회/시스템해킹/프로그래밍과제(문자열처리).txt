회원 한 명에 대해 다음과 같이 처리되는 프로그램을 완성하시오.
 = 회원가입 및 로그인은 프로그램에 실행 중에만 가능하며, 프로그램이 종료되면 기존 회원에 대한 정보가 사라진다.(파일 등으로 저장할 필요 없음)
 = 각 기능마다 함수로 구현할 것.
 = 함수 구현이 불가능한 경우 main 에 모두 작성할 것
 = 그 외 필요하다고 생각되는 추가 기능은 임의대로 추가할 것

1. 회원가입  2. 로그인  0. 종료 :

1. 회원 가입 처리 
 1) 회원가입이 처음이면 ID 및 암호를 입력받는다.(암호는 확인 입력을 하여 정확한 암호 입력을 유도해야 한다)
      ID 입력 :
     => ID는  000 (yes | no) :     (no를 입력하면 "회원가입을 취소하였습니다." 라는 메시지와 함께 메뉴로 돌아간다.)
      PW 입력 : 
      PW 확인 :
    => 암호 두 번 입력이 다른 경우 3번까지 다시 입력할 기회를 준다. 3회까지 암호 입력이 틀리면 다음 메시지와 함께 메뉴로 돌아간다.
           회원가입이 취소되었습니다. 다시 시도해 주세요

 2) 회원가입이 되어 있는 경우
      이미 회원가입이 되어 있습니다. 로그인을 진행하세요
      다시 메뉴화면으로 돌아간다.

2. 로그인 처리
 1) ID, PW를 입력받아서  맞으면 다음과 같은 메시지가 나온다.
    ID 입력 : 
    PW 입력 : 
     000 님 로그인이 성공하였습니다.   

    1. 회원정보 수정  2. 로그아웃   0. 종료 : 
   
   => 로그아웃과 종료만 구현한다. 로그아웃은 이전 메뉴로 돌아가고, 종료는 프로그램이 종료된다.

 2) ID 또는 PW가 틀린경우 다음과 같은 메시지가 나오고 메뉴로 돌아간다.
    : ID 또는 암호가 틀립니다. 다시 확인하시고 로그인해주세요.

 3) 회원가입이 되어 있지 않은 경우 다음과 같은 메시지가 나오고 메뉴로 돌아간다.
    : 회원가입을 먼저 진행해주세요.
