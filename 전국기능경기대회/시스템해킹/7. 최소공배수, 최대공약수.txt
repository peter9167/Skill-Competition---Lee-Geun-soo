최대공약수, 최소공배수

문제(최대공약수, 최소공배수) 해결 단계 (알고리즘 단계)
 1. 문제 분석
 2. 입출력 설계( 입력 세 정수 : a, b, c / 출력 최대공약수 : g <- 세 정수를 나누는 숫자를 곱하여 전체 g변수의 저장 )
입력 후 a,b,c 변수는 계속 나눠지는 몫으로 변경 / 최소공배수 : l <- 세 정수를 나누는 숫자와 서로소를 곱하여 전체 l변수의 저장 ) 
 3. 알고리즘
  - 세 정수 입력
  - 세 정수 값 비교 후 최소값 구하기
  - 반복문으로 i 를 증가시켜 세 정수를 같이 나눌 수 있는 값으로 나눔(정수 값 변경)
